# UniFi Access

Home Assistant integration for **UniFi Access** doors, locks, and sensors.  
This custom integration allows you to monitor and control your UniFi Access devices directly from Home Assistant.

---

## Features

- Monitor door status (open/closed)  
- Lock/unlock doors  
- Battery and sensor monitoring  
- Real-time updates via WebSocket (placeholder for future improvements)
